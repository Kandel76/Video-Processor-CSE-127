To generate oas file, follow the steps below:

Navigate to gf180mcu file and "uv sync"
Then activate the virtual environment with "source .venv/bin/activate"
Go back to oas\_file\_generator, "cd .."
Finally run the python file with "python3 photodiode\_layout.py"
Access and view the resulting layout with "klayout ~/Video-Processor-CSE-127/build/oas/photodiode\_layout.oas"
